## 概述

​  LED是发光二极管的缩写，由混合化合物制成，即镓（Ga），砷（AS），磷（P）。 砷化镓二极管发红光，磷化镓二极管发绿光，氮化镓二极管发蓝光，碳化硅二极管发黄光。发光二极管的反向击穿电压为5V。 其正伏安特性曲线太陡，模块上板载了限流电阻，以便在使用时控制流过管道的电流。
